# 📊 Dashboard de Rendimiento General del Negocio - Retail Analytics

## 📝 Descripción
Este proyecto presenta una solución de **Business Intelligence** desarrollada en Power BI para transformar datos transaccionales en una herramienta estratégica de toma de decisiones. El tablero permite monitorear el estado financiero de una empresa retail mediante la visualización clara de KPIs críticos y tendencias de mercado.

---

## 🚀 Beneficios del Dashboard
* **Monitoreo de KPIs en Tiempo Real**: Visualización inmediata de Ventas Acumuladas, Utilidad Generada, Rentabilidad y Crecimiento.
* **Análisis de Crecimiento Interanual (YoY)**: Identificación de variaciones de ventas con formato condicional dinámico (Verde/Rojo) para una detección rápida de tendencias.
* **Optimización de Categorías**: Desglose detallado por subcategorías para identificar productos estrella y márgenes de ganancia específicos.
---

## 🛠️ Stack Técnico
* **Herramienta**: Power BI Desktop.
* **Lenguaje**: DAX (Data Analysis Expressions) para medidas calculadas de crecimiento y márgenes.
* **Procesamiento**: Power Query para la limpieza y transformación de datos (ETL).

---

## 📱 Vista Móvil (Mobile Optimized)
El dashboard cuenta con una vista específica para smartphones que prioriza la legibilidad vertical:
1. **Jerarquía de KPIs**: Los indicadores más importantes aparecen en la parte superior.
2. **Interactividad**: Filtros adaptados para uso táctil y gráficos de fácil lectura en pantallas pequeñas.
3. **Scroll Eficiente**: Navegación fluida que permite pasar del resumen general al detalle de subcategorías.

---

## 📈 Visualizaciones Incluidas
* **Tarjetas de KPI**: Resumen ejecutivo de métricas principales.
* **Gráfico de Barras**: Comparativa de Ventas vs. Año Anterior.
* **Gráfico de Rosca**: Distribución de ventas por canal (Física vs. En Línea).
* **Tabla de Detalle**: Análisis profundo de utilidad y ventas por subcategoría con iconos de tendencia.

---

## 💡 Personalización
Este tablero es **100% adaptable** a las necesidades de cualquier PyME o empresa, permitiendo integrar metas de ventas específicas, alertas personalizadas y segmentaciones geográficas adicionales.
